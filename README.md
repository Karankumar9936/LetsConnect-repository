# LetsConnect-repository
Let'sConnect 🚀
Bridging the gap between Brands and Nano/Micro-Influencers.

Let'sConnect is a full-stack platform designed to help small-scale influencers find meaningful brand collaborations while providing brands with an easy way to manage campaigns and track performance.

🌟 Key Features
For Influencers
Personalized Dashboard: Track total earnings and active campaign status.

Campaign Discovery: Browse and apply for featured campaigns with "View All" and "Category" filters.

Real-time Messaging: Chat directly with brands once an application is accepted.

For Brands
Campaign Management: Create, edit, and monitor performance of marketing campaigns.

Influencer Verification: View influencer profiles and social media links before hiring.

Secure Payments: Integrated simulation for payout processing.

For Admins
User Management: Approve or remove users to maintain platform quality.

System Oversight: Monitor all active entities and campaign health.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.

Backend: Node.js, Express.js.

Database: MySQL (Relational Schema).

Authentication: JWT (JSON Web Tokens) with Role-Based Access Control (RBAC).

Icons: Lucide-React/Lucide-JS.
